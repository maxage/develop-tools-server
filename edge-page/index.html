一站式新闻热搜聚合服务

📖 API 使用说明

本服务提供多个平台的新闻热搜聚合API，支持实时获取各平台热门内容。

GET 获取平台列表
/platforms
GET 获取新闻数据
/news?platform={platform_name}
GET 图片代理服务
/proxy/image?url={image_url}&w={width}&h={height}

💻 调用示例

JavaScript 调用示例

// 获取平台列表 fetch('/platforms') .then(response => response.json()) .then(data => console.log(data)); // 获取百度热搜 fetch('/news?platform=baidu') .then(response => response.json()) .then(data => console.log(data)); // 获取微博热搜 fetch('/news?platform=weibo') .then(response => response.json()) .then(data => console.log(data));

cURL 调用示例

# 获取平台列表 curl "https://your-domain.com/platforms" # 获取百度热搜 curl "https://your-domain.com/news?platform=baidu" # 获取微博热搜 curl "https://your-domain.com/news?platform=weibo"

Python 调用示例

import requests # 获取平台列表 response = requests.get('https://your-domain.com/platforms') platforms = response.json() # 获取百度热搜 response = requests.get('https://your-domain.com/news?platform=baidu') news = response.json()
